Let’s get started

Screen 1: User can either upload a document or paste text to start the process.

Pick your fragments

    Screen 2: User can highlight keywords and sentences they want to use in the new content, and add a prompt to guide generation.

    What’s your vibe?

    Screen 3: User selects the type of post they want to create from a predefined list.

    Set the tone

    Screen 4: User chooses the tone of voice for the post, which influences the generation style.

    Tweak your post

    Screen 5: User sees the generated post and can edit it or regenerate based on previous inputs.

    Take a peek

    ✨ Tip: Short and punchy lines grab attention!

    Screen 6: User previews the final post with a simple suggestion, and can go back to edit.

    Ready to shine!

    Screen 7: User exports the final post by copying to clipboard or downloading as a text file.
